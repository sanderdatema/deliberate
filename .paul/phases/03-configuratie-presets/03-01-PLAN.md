<objective>
## Goal
Add a configuration system with team presets (quick/balanced/deep), support for custom personas, and the remaining 5 Aslander personas for the "deep" preset. This completes v0.1.

## Purpose
The tool works but is rigid — always 8 personas, always 2 rounds. Users need the ability to choose between a quick analysis (3 min, 4 agents) and a deep dive (10 min, 13+ agents). Custom persona support lets users add domain-specific thinkers.

## Output
- config.yaml with team presets and defaults
- 5 additional personas (Aslander's remaining figures) for "deep" preset
- Updated /deliberate command with --preset flag support
- Custom persona loading from project directory
- Tests for config parsing and new personas
- Updated CLAUDE.md
</objective>

    **Create config.yaml in project root:**
    ```yaml
    default_preset: balanced
    rounds: 2
    model: opus

    presets:
      quick:
        description: "Snelle analyse — 3 analisten + 1 editor, 1 ronde"
        rounds: 1
        analysts: [occam, holmes, lupin]
        editors: [marx]
      balanced:
        description: "Standaard deliberatie — 5 analisten + 3 editors, 2 rondes"
        rounds: 2
        analysts: [socrates, occam, da-vinci, holmes, lupin]
        editors: [marx, hegel, arendt]
      deep:
        description: "Diepgaande analyse — alle 10 analisten + 3 editors, 2 rondes"
        rounds: 2
        analysts: [socrates, occam, da-vinci, holmes, lupin, templar, tubman, weil, marple, noether]
        editors: [marx, hegel, arendt]
    ```

    **Create 5 new analyst personas following the same YAML schema:**

    1. **templar.yaml** (Simon Templar / The Saint):
       - FORBIDDEN from proposing solutions. MUST only reveal power dynamics and hidden interests.
       - FORBIDDEN from taking sides. Must expose who benefits and who pays.
       - Focus: power structures, incentives, cui bono

    2. **tubman.yaml** (Harriet Tubman):
       - FORBIDDEN from accepting theoretical arguments without testing under pressure.
       - MUST test if ideas hold when escape routes are closed.
       - FORBIDDEN from optimism without evidence of resilience.
       - Focus: stress-testing, survival conditions, practical viability

    3. **weil.yaml** (Simone Weil):
       - FORBIDDEN from abstraction. MUST ground everything in concrete, lived observation.
       - FORBIDDEN from comfortable conclusions. Must strip away what is merely reassuring.
       - Focus: concrete reality, attention, removing intellectual comfort

    4. **marple.yaml** (Miss Marple):
       - FORBIDDEN from being impressed by the extraordinary. MUST find the ordinary pattern.
       - FORBIDDEN from accepting context at face value. Things are never as special as they seem.
       - Focus: ordinary patterns in extraordinary situations, village wisdom

    5. **noether.yaml** (Emmy Noether):
       - FORBIDDEN from focusing on what changes. MUST identify what stays the same.
       - FORBIDDEN from surface-level analysis. Must find the deep invariant.
       - Focus: invariants, symmetries, conservation laws of the argument

    All with role: analyst, same FORMAT YOUR RESPONSE section, >= 2 forbidden items.

    **Update tests/test_personas.py:**
    - Change expected count from 8 to 13 (10 analysts + 3 editors)

    **Create tests/test_config.py:**
    - Validate config.yaml parses correctly
    - Check all 3 presets exist with required fields
    - Check all persona references in presets point to existing YAML files
    - Check default_preset is a valid preset name
  </action>
