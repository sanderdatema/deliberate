<objective>
## Goal
Create the Python package foundation: data models (dataclasses) for personas, config, and deliberation events, plus loaders that read the existing YAML files into those models. This is the base layer everything else builds on.

## Purpose
The current orchestration lives entirely in a Claude Code slash command prompt (`deliberate.md`). To enable scripting, CLI access, web UI, and proper testing, we need the domain logic in Python. This plan creates the data layer.

## Output
- `deliberators/` Python package with models and YAML loaders
- Tests proving loaders correctly parse all 14 existing personas + config.yaml
- Fix config/docs mismatch (quick preset: config says 2 editors, docs say 1)
</objective>

    **deliberators/loader.py:**
    - `PersonaLoader` class:
      - `load(path: Path) -> Persona` — reads single YAML, returns Persona dataclass
