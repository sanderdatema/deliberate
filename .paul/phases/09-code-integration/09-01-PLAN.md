       - Each agent prompt includes `CODE UNDER REVIEW:` section with the file contents
       - Samenvatter produces actionable code review summary
       - Meta-analysis focuses on consensus issues, priority fixes, contradictions
    8. **Custom persona discovery:** Standard code personas are: linus, kent-beck, fowler, schneier, jobs, don-norman, jony-ive, christensen, hopper, code-synthesizer (plus all original 15)

    Keep the command lean — reuse orchestration structure from deliberate.md, only change preset selection, context injection, and intake questions.
