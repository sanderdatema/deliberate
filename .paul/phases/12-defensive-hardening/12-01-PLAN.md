<objective>
## Goal
Fix 3 LOW-priority findings from the `/deliberate-code` self-review: add path sanitization + filesize limit in context.py, change mutable `list[str]` to `tuple[str, ...]` in frozen dataclasses, and refactor CodeContextBuilder class to module-level function.

## Purpose
Completes the v0.4 Reliability & Code Quality milestone. These defensive hardening changes prevent subtle bugs (mutable fields in frozen dataclasses), improve security (path traversal, resource exhaustion), and simplify the API (stateless class → plain function).

## Output
- `context.py` with path sanitization, filesize limit, and `build_code_context()` module function
- `models.py` with `tuple[str, ...]` replacing `list[str]` in frozen dataclasses
- `loader.py` updated to construct tuples instead of lists
- Tests covering new defensive behavior
</objective>

<task type="auto">
  <name>Task 1: Path sanitization + filesize limit + class→function refactor in context.py</name>
  <files>deliberators/context.py, deliberators/__main__.py</files>
  <action>
    In context.py:
    1. Add MAX_FILE_SIZE constant (1_048_576 bytes = 1 MB) at module level
    2. Add path sanitization check in the build function: reject paths containing ".." by calling `path.resolve()` and checking it doesn't escape a reasonable boundary. Simplest: skip any path where `".." in path.parts`.
    3. Add filesize check: `path.stat().st_size > MAX_FILE_SIZE` → skip with warning
    4. Rename `CodeContextBuilder.build()` to module-level `build_code_context(paths)` function
    5. Remove the `CodeContextBuilder` class entirely — it only had one `@staticmethod`
    6. Keep `_detect_language`, `_is_binary`, `EXTENSION_MAP` as-is (already module-level)

    In __main__.py:
    - Change import from `from deliberators.context import CodeContextBuilder` to `from deliberators.context import build_code_context`
    - Change call from `CodeContextBuilder.build(args.files)` to `build_code_context(args.files)`
  </action>
  <verify>uv run pytest tests/test_context.py -v</verify>
  <done>AC-1, AC-2, AC-4 satisfied: path traversal rejected, oversized files skipped, class replaced by function</done>
</task>

    3. Change `Preset.editors: list[str]` → `editors: tuple[str, ...]`

    Note: `Persona.output_format: dict[str, bool]` is also mutable in frozen dataclass, but dicts have no frozen equivalent in stdlib. Leave it as-is — it's a known limitation, not worth adding frozendict dependency.
