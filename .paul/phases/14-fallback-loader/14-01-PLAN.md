<objective>
## Goal
Add fallback path resolution to the loader and CLI so that `deliberators` works from any directory — searching CWD first, then `~/.config/deliberators/`, then bundled package data.

## Purpose
Makes `uv tool install .` produce a globally usable CLI. Without fallback logic, the CLI fails when run outside the project directory because `config.yaml` and `personas/` don't exist in CWD.

## Output
- `resolve_config_path()` and `resolve_personas_dir()` functions in `loader.py`
- `__main__.py` defaults use the resolve functions instead of hardcoded CWD paths
- Tests verifying the 3-level fallback chain
</objective>

  <name>Task 1: Add resolve functions to loader.py</name>
  <files>deliberators/loader.py</files>
  <action>
    Add two module-level functions at the top of loader.py (after imports):

    ```python
    from deliberators import get_data_path

    _USER_CONFIG_DIR = Path.home() / ".config" / "deliberators"

    def resolve_config_path(explicit: Path | None = None) -> Path:
        """Resolve config.yaml path with fallback chain: explicit → CWD → user config → bundled."""
        if explicit is not None and explicit != Path("config.yaml"):
            return explicit  # User explicitly specified a path
        # CWD
        cwd_path = Path("config.yaml")
        if cwd_path.exists():
            return cwd_path
        # User config dir
        user_path = _USER_CONFIG_DIR / "config.yaml"
        if user_path.exists():
            return user_path
        # Bundled
        return get_data_path() / "config.yaml"

    def resolve_personas_dir(explicit: Path | None = None) -> Path:
        """Resolve personas directory with fallback chain: explicit → CWD → user config → bundled."""
        if explicit is not None and explicit != Path("personas"):
            return explicit  # User explicitly specified a path
        # CWD
        cwd_path = Path("personas")
        if cwd_path.is_dir():
            return cwd_path
        # User config dir
        user_path = _USER_CONFIG_DIR / "personas"
        if user_path.is_dir():
            return user_path
        # Bundled
        return get_data_path() / "personas"
    ```

    Key design decisions:
    - If the user passes an explicit --config or --personas-dir that differs from the default, use it directly (no fallback)
    - If the path matches the default ("config.yaml" or "personas"), apply fallback chain
    - _USER_CONFIG_DIR is a module constant for testability
