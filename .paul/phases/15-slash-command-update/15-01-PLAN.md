## Goal
Update the global slash commands (`/deliberate` and `/deliberate-code`) to resolve config and persona paths dynamically, so they work from any project directory — not just the debaters repo.

- Global copies at `~/.claude/commands/` updated to match
- End-to-end verification that `/deliberate` works from a non-debaters directory
