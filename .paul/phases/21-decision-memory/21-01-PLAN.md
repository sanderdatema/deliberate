<objective>
## Goal
Add structured JSON storage for deliberation results so users can review past deliberations and start follow-up sessions referencing prior conclusions.

## Purpose
Currently every deliberation is fire-and-forget — output goes to stdout and is lost. Decision memory enables:
- Learning from past deliberations (what was concluded, what was deferred)
- Follow-up deliberations that reference prior context
- Building institutional memory across sessions

## Output
- New `deliberators/storage.py` module with DecisionStore class
- DecisionRecord dataclass for structured storage
- CLI `--history` flag to list past deliberations
- CLI `--followup ID` flag to inject prior conclusions as context
- Auto-save after every deliberation
- Tests for storage, retrieval, and follow-up injection
</objective>
