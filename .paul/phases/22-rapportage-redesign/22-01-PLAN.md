<objective>
## Goal
Redesign the output format from per-persona dumps to a thematic, integrated report that users can directly act on. The new format organizes insights by theme (landscape, tensions, blind spots, action items) rather than by who said what.

## Purpose
Current output is a wall of per-persona text — users must mentally synthesize across 5-8 analysts + 3 editors themselves. A thematic report does this work for them: what's the landscape, where are the tensions, what did nobody question, and what should you do.

## Output
- New `ThematicFormatter` in formatter.py (replaces current format method)
- Meta-analysis section generated by a synthesis functional agent
- Updated slash command to use thematic report format
- Updated formatter tests
</objective>

## AC-1: Thematic report structure
```gherkin
Given a completed deliberation with analyst and editor output
When the formatter generates the report
Then it produces sections in this order:
  1. "Kort & Concreet" (samenvatter, unchanged)
  2. "Deliberatie: {question}" title
  3. "Het Landschap" (synthesis of what the question is really about, key dimensions)
  4. "Spanningsvelden" (where analysts disagree, key tensions)
  5. "Blinde Vlekken" (what nobody questioned, from editors)
  6. "Verschuiving" (how positions moved between rounds, if >1 round)
  7. "Actiepunten" (concrete next steps distilled from all output)
  8. "Volledig Verslag" (collapsible per-persona output, preserved for reference)
```

