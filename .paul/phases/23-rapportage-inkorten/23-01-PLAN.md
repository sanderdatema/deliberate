# Phase 23 Plan 01: Rapportage Inkorten

**Rapport output drastisch verkorten: Volledig Verslag appendix standaard weglaten, alleen tonen met --verbose. Synthese-prompt compacter.**

## Context

De meta-deliberatie op eigen codebase produceerde een rapport dat "zelfs nog vele malen langer" was dan verwacht. Oorzaak: `_format_thematic()` in formatter.py voegt altijd het volledige per-persona appendix toe (Volledig Verslag). Dit is de bulk van de output — elke analist × elke ronde × volledige tekst.

De fix: Volledig Verslag wordt opt-in via `--verbose` flag. Standaard output = Kort & Concreet + thematische synthese secties. Dat is het.

## Acceptance Criteria

| ID | Criterion | Verification |
|----|-----------|-------------|
| AC-1 | Default output bevat GEEN "Volledig Verslag" sectie | `uv run python -m deliberators` output bevat geen `## Volledig Verslag` |
| AC-2 | `--verbose` flag toont Volledig Verslag appendix | `uv run python -m deliberators --verbose` output bevat `## Volledig Verslag` |
| AC-3 | Slash command Step 8 output bevat geen Volledig Verslag | Slash command template geüpdatet |
| AC-4 | Synthesis prompt bevat beknoptheid-constraints | `_SYNTHESIS_SYSTEM_PROMPT` heeft max-length richtlijnen |
| AC-5 | Alle tests passen | `uv run pytest` — 0 failures |

## Tasks

### Task 1: Add --verbose flag to CLI and pass through to formatter

**Action:**
1. In `deliberators/__main__.py`: add `--verbose` argument to `build_parser()`
2. Pass `verbose` bool through to `ResultFormatter.format()`
3. In `deliberators/formatter.py`:
   - Add `verbose: bool = False` parameter to `format()` method
   - In `_format_thematic()`: only include Volledig Verslag appendix when `verbose=True`

**Files:** `deliberators/__main__.py`, `deliberators/formatter.py`

**Done when:** `ResultFormatter.format(result)` returns compact output (no appendix), `ResultFormatter.format(result, verbose=True)` returns full output with appendix.

### Task 2: Update synthesis prompt with conciseness constraints

**Action:**
1. In `deliberators/engine.py`: update `_SYNTHESIS_SYSTEM_PROMPT` to include:
   - "Wees beknopt — maximaal 3-4 alinea's per sectie"
   - "Gebruik opsommingen in plaats van lange paragrafen"
   - "Actiepunten: maximaal 5, elk in 1-2 zinnen"

**Files:** `deliberators/engine.py`

**Done when:** Synthesis prompt has explicit conciseness constraints.

### Task 3: Update slash command report format

**Action:**
1. In `.claude/commands/deliberate.md`: update Step 8 to remove Volledig Verslag section from default output
2. Add note that `--verbose` shows full per-persona output (for CLI only — slash command always shows compact)

**Files:** `.claude/commands/deliberate.md`

**Done when:** Step 8 template shows only Kort & Concreet + synthesis sections, no per-persona appendix.

### Task 4: Update tests for verbose flag

**Action:**
1. In `tests/test_cli.py`: update existing formatter tests to verify no appendix by default
2. Add test that `verbose=True` includes appendix
3. Add test for `--verbose` CLI flag parsing
4. Run full test suite

**Files:** `tests/test_cli.py`

**Done when:** `uv run pytest` — all tests pass, 0 failures.

## Risks

| Risk | Mitigation |
|------|-----------|
| Slash command can't pass --verbose | Slash command always shows compact; --verbose is CLI-only feature |
| Tests hardcoded to expect appendix | Update assertions to match new default behavior |
